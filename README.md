# Spring Boot CLI

A command-line tool to generate Spring Boot projects interactively, similar to create-react-app.

## Installation
```bash
npm install -g springboot-cli
```

## Usage
```bash
springboot-cli create my-app
```

## Features

- Interactive project configuration
- Support for Maven and Gradle
- Multiple database options
- Spring Boot best practices
- AI-powered code generation (optional)

## Development
```bash
npm install
npm start
```
